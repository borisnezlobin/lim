Hello!

In order to build the source code, simply run 
```sh
npm install
```
followed by
```sh
npm run build
```

This will generate the `/dist` directory, which, when zipped, will match the included `Archive.zip`. This project was built with the following:

- MacOS Sonoma 14.5
- 16GB RAM
- NPM 9.6.7
- Node 18.17.1
