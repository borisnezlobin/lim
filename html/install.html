Welcome to Lim!t

Lim will vigilantly lock down websites you've set limits for. You can start simply by just typing in the name of websites to set limits. Or, take a more advanced approach by matching RegEx — the choice is yours!

Lim uses RegEx (Regular Expressions) to know which websites to block. RegEx is a powerful tool! You can use it to match patterns in text, and Lim uses it to match patterns in URLs. This means that you can block a whole range of websites with just one RegEx pattern. Or, you can be more specific and block just one website. If you already know RegEx, you can start typing in your patterns right away. If you don't, don't worry! You can learn more about RegEx at:

  • IceWarp's tutorial
  • Wikipedia
  • MDN (which is more focused on RegEx in JavaScript)
  • Regex101 (a tool to test your RegEx patterns)
Or, if you prefer an easier approach, Lim has some common RegEx patterns made for you. Check out Lim's RegEx Limits to see some examples. Also, you can use Lim's Simple Mode when you're creating limits — this will give you a very simple way to block websites without having to learn RegEx. Welcome to Lim!
