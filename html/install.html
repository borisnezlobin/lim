Welcome to Lim!t

Lim will vigilantly lock down websites you've set limits for. You can start simply by just typing in the name of websites to set limits. Or, take a more advanced approach by matching RegEx — the choice is yours!

Getting Started

First, we need to make sure you have Lim installed. If you're seeing this page, you're already halfway there! You've installed Lim! Now, we need to make sure Lim is running. In order for Lim to do its job, it needs to run in private windows. Otherwise, Lim won't be able to block websites in Private Browsing (which will be enabled if you have history disabled). To make sure Lim has the permissions it needs, follow these steps:

  1. Type about:addons in the address bar and press Enter. This will open Firefox's Add-Ons settings page.
  2. Find Lim in the list of Add-Ons and click on it.
  3. Make sure "Run in Private Windows" is checked.

You'll notice that Lim needs permissions to access your data for all websites. This is scary — but don't worry. Lim is entirely open source on GitHub, meaning that you can see exactly the code that is running. Lim needs these permissions so it can know how much time you've spent on websites. No data ever leaves your device, and everything except your limits is cleared every day.

Now, Lim is up and running! If you don't see the Lim icon in your toolbar, you can click the puzzle piece icon in the toolbar, right-click on Lim, and select "Pin to Toolbar".

Setting Limits

Now that Lim is installed and running, you can start setting limits! The easiest way to set limits is to use Lim's Simple Mode. If you're looking for something extra, you can disable Simple Mode to use full-on RegEx. To set limits, click on the Lim icon in your toolbar and click "Add Limit" — the form just needs a limit name, a RegEx or list of websites, and a time limit. Check out the Simple Mode guide for more information.

Always be careful about setting limits on communication platforms — in the case of an emergency, you won't be able to delete the limit until the next day. Lim is not responsible for any harm caused by blocking emergency communications.

Deleting Limits

Want to delete a limit? Lim won't let you! This is a feature, not a bug. If you could delete a limit, you could just delete it and bypass the limit, right? Lim calls this Delayed Delete — any limit you delete will only be deleted the next day. This means that if you set a limit for 1 hour, you'll have to wait until the next day to delete it. This is a great way to make sure you're sticking to your limits! If you really need to delete a limit, you'll have to disable or uninstall Lim.
