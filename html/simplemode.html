Simple Mode

Simple Mode is a way to create limits without having to learn RegEx. You can use Simple Mode to block websites by typing in the name of the website. For example, if you want to block Facebook, you can just type in "facebook.com" and Lim will block it for you. You can also use Simple Mode to block subdomains, like "www.facebook.com". Simple Mode is a great way to get started with Lim, and you can always switch to RegEx if you want to create more advanced limits.

Overview

To use Simple Mode, just type in the name of the website for the Website Name field! If you want to make a limit that blocks multiple websites (for example, a "Social Media" limit that blocks Twitter, Facebook, and Reddit), you can separate the websites with a comma. For the example, you'd write 

twitter, facebook,reddit
Note that the websites are separated by a comma, but spaces don't matter — Lim will remove them for you. Also note that you don't need to include the www or .com. Lim does this on purpose — many websites have multiple subdomains (like mobile.twitter.com) and multiple top-level domains (like facebook.co.uk). Lim's Simple Mode will handle these cases for you!

Examples

If you need to block a specific subdomain, you can do that too! For example, if you want to block Google Drive, you can just type in drive.google and Lim will block it for you. Here are some ready-to-use examples:

Communications

Be careful about blocking emergency communications — if Lim blocks a web page, you won't be able to delete the limit until the next day. This can be dangerous. Proceed with caution. Always ensure you have a way to contact emergency services and contacts. Lim is not responsible for any harm caused by blocking emergency communications.
